
# Finance Dashboard Backend

## Features
- Role-based access control
- Financial records CRUD
- Dashboard analytics
- Pagination & filtering
- Search support
- Soft delete

## Tech Stack
Node.js, Express, MongoDB

## Setup
npm install
npm run dev

## API Docs
/api-docs

## Roles
- Viewer: Read only
- Analyst: Read + dashboard
- Admin: Full access

## Assumptions
- Single user system
- JWT auth used
- No multi-tenant support
